# San Francisco Hotels & Neighborhoods Map

An interactive map showing hotel options and desirable neighborhoods in the San Francisco Bay Area for a stay from November 28-30, 2025.

## Usage

Simply open `index.html` in your web browser to view the interactive map. Click on any marker to see details, and click hotel names to visit their websites.

## View Online

To host this on GitHub Pages:
1. Go to your repository settings
2. Navigate to "Pages" 
3. Select your main branch as the source
4. Your map will be available at: `https://[your-username].github.io/SFhotels/`
